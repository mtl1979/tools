Administrator Commands

/deop [name or session id]

Function: take admin status from other user
See also: op

/invite [name or session id]

Function:  invite user to channel
See also: kick

/kick [name or session id]

Function: kick user off the channel
See also: invite

/op [name or session id]

Function: give admin status to other user
See also: deop

/private

Function: set channel to private mode
See also: public

/public

Function: set channel to public mode
See also: private

/topic [topic]

Function: change the channel topic
