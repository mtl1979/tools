Administrator Commands

/addbans [users]

Function: add bans by user names or session ids
See also: rembans

/addipbans [ips]

Function: add bans by ip addresses
See also: remipbans

/kick [users]

Function: kick by user names or session ids

/kickips [ips]

Function: kick by ip addresses

/rembans [users]

Function: remove bans by user names
See also: addbans

/remipbans [ips]

Function: remove bans by ip addresses
See also: addipbans

/require [ips]

Function: add require mask
Comments: Only users coming from specified "require masks" can enter server, if atleast one mask is specified
See also: unrequire

/unrequire [ips]

Function: remove require mask
See also: require
