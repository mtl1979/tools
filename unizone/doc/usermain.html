User Commands

/action [action]

Aliases: /me

Function: do something

/addautopriv [pattern]

Function: update the auto-private pattern (can be a user name, or several names, or regular expression)
See also: autopriv, priv, unautopriv

/addblacklist [pattern]

Function: update the blacklist pattern (can be a user name, or several names, or regular expression)
See also: blacklist, unblacklist

/addignore [pattern]

Function: update the ignore pattern (can be a user name, or several names, or a regular expression)
See also: ignore, unignore

/autopriv [pattern]

Function: set the auto-private pattern (can be a user name, or several names, or regular expression)
See also: addautopriv, priv, showpatterns, unautopriv

/away

Function: set away state (same as selecting away from the list)
See also: awaymsg

/awaymsg [message]

Function: away message for away state (when /away is invoked)
See also: away, heremsg

/blacklist [pattern]

Function: set the blacklist pattern (can be a user name, or several names, or a regular expression)
See also: addblacklist, showpatterns, unblacklist

/btime [gmt]

Parameters: gmt (optional) - Show GMT time, otherwise local time
Function: Broadcast and show time
See also: time

/buptime

Function: Broadcast and show uptime
See also: uptime

/chkserver [index]

Function: check server string
See also: chkstatus, chkuser, remserver, server

/chkstatus [index]

Function: check status string
See also: chkserver, chkuser, remstatus, status

/chkuser [index]

Function: check nick string
See also: chkserver, chkstatus, nick, remuser

/clear

Function: clear the text in the chat view

/clearline

Function: clear all the line buffers

/clearstats

Function: clear transfer statistics
See also: showstats

/compression [level]

Parameters: level (optional) - Set compression level, otherwise view
Function: set or view message compression level
Comments: Requires MUSCLE 2.40 or later
See also: server

/connect

Function: connect to the currently selected server
See also: disconnect, onconnect

/disconnect

Function: disconnect from server
See also: connect

/dns [user|host]

Function: give information about user or host

/help [command]

Parameters: command (optional) - Show help for command, otherwise full command reference
Function:show help for command (no '/' in front of command) or show full command referenceif no command given.

/heremsg [message]

Function: Set message for here state
See also: awaymsg

/ignore [pattern]

Function: set the ignore pattern (can be a user name, or several names, or a regular expression)
See also: addignore, unignore

/logged

Function: show the time you have been logged in to a server
See also: running, uptime

/msg [name] [message]

Function: send a private message

/nick [name]

Function: change your user name

/onconnect [command]

Parameter: command (optional) set command, otherwise clear
Function: set or clear command to perform on successful connect
See also: connect, showpatterns

/ping [name or session ids]

Function: ping other clients

/priv [name or session ids]

Function: open private chat with these users added
See also: addautopriv, autopriv, unautopriv

/quit

Function: quit Unizone

/remote [password]

Parameter: password (optional) set password, otherwise view
Function: set or view remote password

/remserver [index]

Function: remove server from server list
See also: chkserver, server

/remstatus [index]

Function: remove status from status list
See also: chkstatus, status

/remuser [index]

Function: remove nick from nick list
See also: chkuser, nick

/resumes

Function: list files waiting to be resumed

/running

Function: show time elapsed since you started Unizone
See also: logged, uptime

/save

Function: saves settings
Comments: (might be necessary after editing drop-down lists)

/scan

Function: rescan shared directory

/search [pattern]

Function: open search pane and start query for pattern

/server [server]

Function: set the current server
See also: chkserver, compression, remserver, serverinfo

/serverinfo

Function: check status of server
See also: server

/shell [command]

Function: execute command

/showpatterns

Function: show auto-private, blacklist, ignore and watch patterns
See also: autopriv, blacklist, ignore, onconnect, watch

/showstats

Function: show transfer statistics
See also: clearstats

/status [status]

Function: set status string
See also: chkstatus, remstatus

/time [gmt]

Parameters: gmt (optional) - show GMT time, otherwise local time
Function: show time
See also: btime

/time [nick] [gmt]

Parameters: gmt (optional) - request GMT time stamp, otherwise local time stamp
Function: request time stamp from other user

/unautopriv [name]

Function: remove name from auto-private list
See also: addautopriv, autopriv

/unblacklist [name]

Function: remove name from blacklist
See also: addblacklist, blacklist

/unignore [name]

Function: remove name from ignore list
See also: addignore, ignore

/uptime

Function: show system uptime
See also: buptime, logged, running

/users

Function: show number of users connected

/version

Function: show client version strings

/watch [pattern]

Function: set the watch pattern (can be a user name, or several names, or a regular expression)
See also: showpatterns
