To add a new language:

1. Add entry in src/unizone.pro
2. Run 'lupdate unizone.pro'
3. Run 'linguist unizone_xx.ts', where xx is the two letter language code

NOTE:	You need to use special version of lupdate/linguist included in the package. 
	Standard versions use different character encoding with temporary (*.ts) files.
